# Ds-ai-portofolio
Modern &amp; Interactive Data Science Portfolio Website. Built with Vanilla JS, Canvas API, and Glassmorphism CSS. Featuring AI-themed animations and responsive design.  Topics: data-science, portfolio, vanilla-javascript, css-animations, ai-trainer, web-development, glassmorphism.

Sebuah website portfolio modern, responsif, dan interaktif yang dirancang khusus untuk profesional di bidang **Data Science** dan **AI Training**. Website ini dibangun dengan fokus pada estetika "futuristic-dark" menggunakan teknologi web standar tanpa framework berat untuk performa maksimal.
